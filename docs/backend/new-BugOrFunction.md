# bug反馈 & 功能建议

### bug反馈
> 所有bug在确认issues中没有相同类型问题后，请附带操作环境、软件版本、爬取链接以及详细描述复现过程后(如果有报错日志，请贴日志)，统一提交至[Issues](https://github.com/xisuo67/Unbelievable-Box/issues)，提交格式如下：

```
是否搜索过Issues，并且没有找到相关解决方案：是/否
操作环境：xxx(自己操作环境)
软件版本：v1.0.1.3/x.x.x.x
链接：xxxxx
复现过程：xxxxxx

```

::: tip 提示
 很少上QQ，有问题直接提交Issues
:::
